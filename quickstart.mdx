<Info>
  **Prerequisites**: A ClubSpeed account with API access enabled. Contact your account manager if you need access.
</Info>

<Steps>
<Step title="Get your API key">
  Navigate to **Settings > API Keys** in your ClubSpeed dashboard. Create a new key and copy the value.

  - **Live keys** start with `sk_live_` and access production data.
  - **Test keys** start with `sk_test_` and access the sandbox environment.

  <Warning>Store your API key securely. Do not commit it to version control or expose it in client-side code.</Warning>
</Step>

<Step title="Make your first request">
  Use your API key to list customers:

  <CodeGroup>
  ```bash cURL
  curl https://api.platform.io/v1/customers \
    -H "Authorization: Bearer sk_test_your_key_here"
  ```

  ```javascript Node.js
  const response = await fetch("https://api.platform.io/v1/customers", {
    headers: {
      "Authorization": "Bearer sk_test_your_key_here"
    }
  });
  const data = await response.json();
  console.log(data);
  ```

  ```python Python
  import requests

  response = requests.get(
      "https://api.platform.io/v1/customers",
      headers={"Authorization": "Bearer sk_test_your_key_here"}
  )
  data = response.json()
  print(data)
  ```
  </CodeGroup>
</Step>

<Step title="Inspect the response">
  A successful response returns a paginated list:

  ```json
  {
    "object": "list",
    "url": "/v1/customers",
    "has_more": true,
    "data": [
      {
        "id": "cus_abc123",
        "object": "customer",
        "first_name": "Jane",
        "last_name": "Doe",
        "email": "jane@example.com",
        "created_at": "2025-01-15T08:30:00Z"
      }
    ]
  }
  ```
</Step>
</Steps>
